# GuidesTable

Render a table or grid of guides with optional caption and filters.

## When to use

- Use to render guide listings and card-based navigation for documentation.

## Props

- `caption` - Optional caption text.
- `guides: initialGuides` - Optional prop.
- `subcategory` - Optional prop.
- `subdirectory` - Optional prop.
- `subdirectoryGuides: propGuides` - Optional prop.

## Example

```jsx
<GuidesTable guides={[]} caption="Available guides" />
```
