# CompoundBreakdown

Render a breakdown of a chemical compound by formula and parts.

## Props

- `className` — Additional CSS classes to apply.
- `formula` — Optional prop.
- `parts` — Optional prop.

## Example

```jsx
<CompoundBreakdown formula="H2O" parts={[{symbol: "H", count: 2}, {symbol: "O", count: 1}]} />
```
