# Figure

Render an image figure with optional caption and width control.

## When to use

- Use when an image needs a caption and semantic markup.

## Props

- `alt` — Alternative text for accessibility.
- `caption` — Optional caption text.
- `src` — Image source URL.
- `width` — Width value or layout width.

## Example

```jsx
<Figure src="/path/image.png" alt="Example" caption="Image caption" width="100%" />
```

## Notes

Use this component when you need an image with a caption and width control inside content.
