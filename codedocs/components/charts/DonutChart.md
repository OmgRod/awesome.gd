# DonutChart

Render a donut chart with optional legend.

## Props

- `className` - Additional CSS classes to apply.
- `data` - Data points used to render the component.
- `showLegend` - Whether to render a chart legend.
- `size` - Optional prop.
- `strokeWidth` - Optional prop.

## Example

```jsx
<DonutChart data={[{label: "A", value: 30}, {label: "B", value: 70}]} />
```
