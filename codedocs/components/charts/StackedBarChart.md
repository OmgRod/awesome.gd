# StackedBarChart

Render a stacked bar chart for grouped data.

## Props

- `className` - Additional CSS classes to apply.
- `data` - Data points used to render the component.
- `series` - Optional prop.
- `showLegend` - Whether to render a chart legend.

## Example

```jsx
<StackedBarChart data={[{label: "Q1", value: [10, 20]}]} series={["A", "B"]} />
```
