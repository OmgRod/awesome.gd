# StackedBarChart

Render a stacked bar chart for grouped data.

## Props

- `className` ‚Äî Additional CSS classes to apply.
- `data` ‚Äî Data points used to render the component.
- `series` ‚Äî Optional prop.
- `showLegend` ‚Äî Whether to render a chart legend.

## Example

```jsx
<StackedBarChart data={[{label: "Q1", value: [10, 20]}]} series={["A", "B"]} />
```
