# BarChart

Render a vertical bar chart from numeric data.

## Props

- `className` — Additional CSS classes to apply.
- `data` — Data points used to render the component.
- `height` — Height value or layout height.
- `showValues` — Whether to render values on bars.

## Example

```jsx
<BarChart data={[{value: 10}, {value: 20}]} />
```
