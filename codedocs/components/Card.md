# Card

Render a simple content card with title, description, and body.

## When to use

- Use for self-contained content blocks with a title and body.

## Props

- `children` — Nested content or child elements.
- `description` — Secondary descriptive text.
- `title` — Title or heading text.

## Example

```jsx
<Card title="Card title" description="Short summary.">Card details.</Card>
```
