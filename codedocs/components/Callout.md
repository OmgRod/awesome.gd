# Callout

Render a highlighted callout for important notes and tips.

## When to use

- Use to highlight tips, notes, or warnings in documentation content.

## Props

- `children` — Nested content or child elements.
- `icon` — Optional prop.
- `title` — Title or heading text.
- `variant` — Visual variant or style option.

## Example

```jsx
<Callout title="Note" icon="idea">This is a callout.</Callout>
```
