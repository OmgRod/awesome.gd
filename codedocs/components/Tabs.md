# Tabs

Render the Tabs component.

## Usage

Use `<TabPanels>` with nested `<Tab title="...">` children to render tabbed content.

## Props

- `children` — Nested content or child elements.

## Example

```jsx
<TabPanels>
  <Tab title="First tab">First content</Tab>
  <Tab title="Second tab">Second content</Tab>
</TabPanels>
```

## Notes

The default export is `<TabPanels>`. Use nested `<Tab title="...">...</Tab>` children to define each panel.
