# FormulaList

Render a list of formulas with names and expressions.

## When to use

- Use to display a collection of named math formulas.

## Props

- `className` - Additional CSS classes to apply.
- `items` - Array of items or content entries.

## Example

```jsx
<FormulaList items={[{name: "Quadratic Formula", expression: "x = (-b ± √(b²-4ac))/(2a)"}]} />
```
