# Alert

Render a styled alert box for informative, warning, or error messages.

## When to use

- Use to draw attention to important information, warnings, or success messages.

## Props

- `children` - Nested content or child elements.
- `title` - Title or heading text.
- `type` - Optional prop.

## Example

```jsx
<Alert type="info" title="Heads up!">Content goes here.</Alert>
```

## Notes

Use the `type` prop to switch between info, caution, error, note, and success styles.
