# PresetSearch

Render a searchable list of preset items.

## Props

- `emptyMessage` - Message displayed when no results are found.
- `items` - Array of items or content entries.
- `limit` - Maximum number of items to display.
- `placeholder` - Placeholder behavior for inputs or image loading.

## Example

```jsx
<PresetSearch items={[]} placeholder="Search presets..." />
```
