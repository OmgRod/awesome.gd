# PresetSearch

Render a searchable list of preset items.

## Props

- `emptyMessage` ‚Äî Message displayed when no results are found.
- `items` ‚Äî Array of items or content entries.
- `limit` ‚Äî Maximum number of items to display.
- `placeholder` ‚Äî Placeholder behavior for inputs or image loading.

## Example

```jsx
<PresetSearch items={[]} placeholder="Search presets..." />
```
