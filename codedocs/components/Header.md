# Header

Render the top site header and navigation.

## When to use

- Use as the top header and primary navigation section of the site.

## Props

- `headerConfig` — Header configuration data.
- `searchDocuments` — Documents used as search targets.
- `siteConfig` — Site configuration data.

## Example

```jsx
<Header headerConfig={{}} searchDocuments={[]} siteConfig={{}} />
```
