# AuthorCard

Render an author profile card with avatar, name, and link.

## Props

- `avatar` - Optional prop.
- `name` - Optional prop.
- `showLabel` - Optional prop.
- `url` - Optional prop.
- `username` - Optional prop.

## Example

```jsx
<AuthorCard username="rodri" name="Rodri" avatar="/avatar.png" url="/author/rodri" />
```
