# ButtonLink

Render a link with button styling for primary actions.

## When to use

- Use for links that should appear as buttons in the UI.

## Props

- `children` — Nested content or child elements.
- `href` — Destination URL for a link.
- `variant` — Visual variant or style option.

## Example

```jsx
<ButtonLink href="/download" variant="primary">Download</ButtonLink>
```
